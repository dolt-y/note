# 常见终端命令

## ls

显示当前目录下的文件和目录。

```
ls
```

## cd

切换目录。

```
cd /path/to/directory
```

## pwd

显示当前目录的路径。

```
pwd
```

## mkdir

创建目录。

```
mkdir directory_name
```

## touch

创建文件。

```
touch file_name
```

## rm

删除文件或目录。

```
rm file_name
```

## mv

移动或重命名文件或目录。

```
mv old_file_name new_file_name
```

## cp

复制文件或目录。

```
cp old_file_name new_file_name
```

## cat

显示文件内容。

```
cat file_name
```

## grep

搜索文件内容。

```
grep "search_string" file_name
```

## man

显示命令的帮助信息。

```
man command_name
```

## chmod

修改文件或目录的权限。

```
chmod 777 file_name
```

## chown

修改文件或目录的拥有者。

```                
chown user_name file_name
```

## vi
编辑文件。

```
vi file_name
```
