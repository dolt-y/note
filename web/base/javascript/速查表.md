# JavaScript 快速参考手册

## 数组操作

| 技巧        | 示例                         | 说明                         |
| --------- | -------------------------- | -------------------------- |
| **map**   | `[1,2,3].map(x => x*2)`     | 遍历数组并返回新数组            |
| **filter**| `[1,2,3].filter(x => x>1)`  | 筛选符合条件的元素              |
| **reduce**| `[1,2,3].reduce((a,b)=>a+b,0)` | 数组汇总、累加、扁平化           |
| **find**  | `[1,2,3].find(x => x>1)`    | 找到第一个符合条件的元素          |
| **some / every** | `[1,2,3].some(x>2)`      | 检查部分或全部元素是否符合条件       |
| **展开合并** | `[...arr1, ...arr2]`       | 合并多个数组                   |
| **去重**   | `[...new Set(arr)]`         | 快速去重数组                   |
| **数组转对象** | `Object.fromEntries(arr.map(i => [i.key,i.value]))` | 根据键值数组生成对象            |

## 对象操作

| 技巧                | 示例                             | 说明                    |
| ----------------- | ------------------------------ | --------------------- |
| **属性解构**         | `let {a,b} = obj`               | 快速获取对象属性           |
| **默认值解构**        | `let {x=0,y=0} = obj`           | 防止 undefined           |
| **展开合并**         | `{...obj1, ...obj2}`            | 合并对象或浅拷贝           |
| **动态属性**         | `[key]: value`                  | 根据变量动态创建属性        |
| **Object.entries/keys/values** | `Object.entries(obj).forEach(([k,v])=>{})` | 遍历对象键值             |
| **可选链**           | `obj?.prop?.sub`                | 链式访问防止报错           |

---

## 函数与参数

| 技巧           | 示例                     | 说明                         |
| ------------- | ---------------------- | -------------------------- |
| **箭头函数**    | `const fn = x => x*2`    | 简洁写法，绑定外层 `this`     |
| **默认参数**    | `fn(x=0)`               | 参数未传递时使用默认值           |
| **剩余参数**    | `function sum(...nums)` | 可变参数求和或批量处理           |
| **解构参数**    | `function fn({a,b})`    | 参数对象解构，更直观             |
| **高阶函数**    | `arr.map(fn).filter(fn)` | 函数组合，链式调用               |
| **柯里化**      | `const add = x => y => x+y` | 延迟计算，函数复用             |

---

## 异步与 Promise

| 技巧              | 示例                                               | 说明                       |
| ---------------- | ------------------------------------------------ | ------------------------ |
| **Promise**       | `new Promise((res, rej) => {})`                  | 创建异步操作                 |
| **then / catch**  | `fetch(url).then(res => res.json()).catch(err => {})` | 链式处理异步                 |
| **async / await** | `let data = await fetchData()`                   | 同步风格处理异步              |
| **Promise.all**   | `await Promise.all([p1, p2])`                    | 并行执行多个异步操作           |
| **Promise.race**  | `await Promise.race([p1, p2])`                   | 谁先完成就用谁的结果           |
| **超时控制**      | `Promise.race([fetch(), timeout()])`            | 异步操作加超时                |

## 模块化与工具

| 技巧        | 示例                             | 说明                  |
| ---------- | ------------------------------ | ------------------- |
| **导出**    | `export const a = 1`           | 模块拆分               |
| **导入**    | `import {a} from './file'`     | 引入模块               |
| **默认导出** | `export default function(){}`  | 模块默认功能            |
| **动态导入** | `import('./file.js').then(m => m.fn())` | 懒加载模块           |
| **内置工具** | `Math, JSON, Date, URL`       | ES6+ 常用方法           |

## 前端优化与辅助技巧

| 技巧           | 示例                     | 说明                              |
| ------------- | ---------------------- | -------------------------------- |
| **防抖**       | `debounce(fn, 300)`    | 减少高频操作触发，如输入框           |
| **节流**       | `throttle(fn, 200)`    | 控制函数执行频率，如滚动             |
| **可迭代对象** | `for(const item of arr)` | 遍历数组、Set、Map                 |
| **Map / Set**  | `let m = new Map(); let s = new Set();` | 高效存储键值对、唯一集合           |
| **Symbol**     | `let id = Symbol('id')` | 创建唯一标识，避免对象属性冲突       |
| **模板字符串** | `` `Hello ${name}` ``  | 拼接动态文本，增强可读性             |
